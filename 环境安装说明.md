## 🚀 启动服务

### 方式一：使用 PowerShell 脚本（推荐）

```powershell
# 启动后端服务
.\start-backend.ps1

# 启动前端服务
.\start-frontend.ps1
```

### 方式二：手动启动

#### 1. 启动后端服务

```bash
# 进入后端目录
cd baiyu-ruoyi

# 启动主应用
cd ruoyi-admin-wms
mvn spring-boot:run

# 或者直接运行 jar 包
java -jar target/ruoyi-admin-wms.jar
```

#### 2. 启动前端服务

```bash
# 进入前端目录
cd baiyu-web

# 启动开发服务器
npm run dev
```

## 📝 访问地址

## 🔧 常见问题

### 1. 后端启动失败
- 检查数据库连接配置
- 检查 Redis 连接配置
- 查看日志：`baiyu-ruoyi/ruoyi-admin-wms/logs/`

### 2. 前端无法访问后端
- 检查后端服务是否启动
- 检查跨域配置
- 检查 API 基础 URL 配置

### 3. 端口被占用
修改配置文件中的端口号即可
