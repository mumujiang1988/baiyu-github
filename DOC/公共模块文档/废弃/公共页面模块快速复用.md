### 📋 使用方法

#### 基础用法
```powershell
# 进入模板目录
cd d:\baiyuyunma\By-web\src\views\k3\pageTemplate

# 执行复制脚本
.\copy-module-v2.ps1 -ModuleName "purchaseOrder" -ModuleTitle "采购订单管理"
```

#### 参数说明
- `-ModuleName`: 模块名称（小写，如 `purchaseOrder`）
- `-ModuleTitle`: 模块标题（中文，如 `采购订单管理`）

###  常见错误处理

#### 错误 1: 目标目录已存在
```
ERROR: Destination folder already exists: d:\baiyuyunma\By-web\src\views\k3\saleOrder
```
**解决方案**: 删除已存在的目录或选择其他模块名称
```powershell
Remove-Item -Path "d:\baiyuyunma\By-web\src\views\k3\saleOrder" -Recurse -Force
```

#### 错误 2: 源路径不存在
```
ERROR: Source path not found: d:\baiyuyunma\By-web\src\views\k3\pageTemplate
```
**解决方案**: 确保当前目录在 `pageTemplate` 下
```powershell
cd d:\baiyuyunma\By-web\src\views\k3\pageTemplate
```

#### 错误 3: 参数缺失
```
Parameter 'ModuleName' cannot be null
```
**解决方案**: 提供完整的参数
```powershell
.\copy-module-v2.ps1 -ModuleName "yourModule" -ModuleTitle "你的模块标题"
```
