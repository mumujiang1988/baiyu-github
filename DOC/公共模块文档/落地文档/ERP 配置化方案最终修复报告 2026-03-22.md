### 5. 数据库优化SQL脚本 

**修复文件**: `ERP 配置化方案数据库优化脚本 2026-03-22.sql`

**优化内容**:
-  添加3个外键约束
-  添加5个复合索引
-  添加2个全文索引
-  补充2个触发器(INSERT/DELETE)

**外键约束**:
```sql
-- erp_page_config_history 外键
ALTER TABLE erp_page_config_history 
ADD CONSTRAINT fk_history_config 
FOREIGN KEY (config_id) REFERENCES erp_page_config(config_id)
ON DELETE CASCADE;

-- erp_approval_history 外键
ALTER TABLE erp_approval_history 
ADD CONSTRAINT fk_history_flow 
FOREIGN KEY (flow_id) REFERENCES erp_approval_flow(flow_id)
ON DELETE SET NULL;

-- erp_page_config 父配置外键
ALTER TABLE erp_page_config
ADD CONSTRAINT fk_parent_config 
FOREIGN KEY (parent_config_id) REFERENCES erp_page_config(config_id)
ON DELETE SET NULL;
```

**复合索引**:
```sql
-- erp_approval_history 复合索引
ALTER TABLE erp_approval_history 
ADD INDEX idx_module_bill (module_code, bill_id);

ALTER TABLE erp_approval_history
ADD INDEX idx_bill_action (bill_id, approval_action);

-- erp_page_config 复合索引
ALTER TABLE erp_page_config
ADD INDEX idx_module_status_version (module_code, status, version);

-- erp_push_relation 复合索引
ALTER TABLE erp_push_relation
ADD INDEX idx_source_status (source_module, status);
```

**全文索引**:
```sql
ALTER TABLE erp_page_config 
ADD FULLTEXT INDEX ft_config_content (config_content);

ALTER TABLE erp_approval_flow
ADD FULLTEXT INDEX ft_flow_definition (flow_definition);
```

**补充触发器**:
```sql
-- INSERT触发器

AFTER INSERT ON `erp_page_config`
FOR EACH ROW
BEGIN

END

-- DELETE触发器
CREATE TRIGGER `trg_erp_config_history_delete`
BEFORE DELETE ON `erp_page_config`
FOR EACH ROW
BEGIN
  INSERT INTO erp_page_config_history (...) VALUES (...);
END
```
