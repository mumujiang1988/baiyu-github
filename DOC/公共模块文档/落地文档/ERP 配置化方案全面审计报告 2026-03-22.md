** 问题1: 缺少外键约束**

**建议修复**:
```sql

ADD CONSTRAINT fk_history_config

ALTER TABLE erp_approval_history 
ADD CONSTRAINT fk_history_flow 
FOREIGN KEY (flow_id) REFERENCES erp_approval_flow(flow_id)
ON DELETE SET NULL;

ALTER TABLE erp_page_config
ADD CONSTRAINT fk_parent_config 
FOREIGN KEY (parent_config_id) REFERENCES erp_page_config(config_id)
ON DELETE SET NULL;
```

** 问题2: 索引覆盖不足**

`erp_approval_history` 表缺少复合索引,影响查询性能。

**建议修复**:
```sql
-- 添加复合索引
```

** 问题3: JSON字段未做大小限制**

`config_content`、`mapping_rules` 等LONGTEXT字段可能存储超大JSON,影响性能。

**建议**:
- 在应用层添加JSON大小校验(建议<1MB)

**修复方案**:

**方案1: 使用Rhino沙箱(推荐)**
```java
import org.mozilla.javascript.*;

public boolean evaluateCondition(String condition, Map<String, Object> context) {
    Context cx = Context.enter();
    try {
        // 设置JavaScript版本
        cx.setLanguageVersion(Context.VERSION_ES6);
        
        // 禁止访问Java类(沙箱)
        cx.setClassShutter(className -> false);
        
        // 创建作用域
        Scriptable scope = cx.initStandardObjects();
        
        // 绑定安全的上下文变量
        for (Map.Entry<String, Object> entry : context.entrySet()) {
            Object jsValue = Context.javaToJS(entry.getValue(), scope);
            ScriptableObject.putProperty(scope, entry.getKey(), jsValue);
        }
        
        // 执行条件表达式
        Object result = cx.evaluateString(scope, condition, null, 1, null);
        
        return Context.toBoolean(result);
    } catch (Exception e) {
        log.error("条件表达式执行失败: {}", condition, e);
        return false;
    } finally {
        Context.exit();
    }
}
```

**方案2: 使用Spring Expression Language (SpEL)**
```java
import org.springframework.expression.*;
import org.springframework.expression.spel.*;
import org.springframework.expression.spel.support.*;

public boolean evaluateCondition(String condition, Map<String, Object> context) {
    try {
        // 创建表达式解析器
        ExpressionParser parser = new SpelExpressionParser();
        
        // 创建上下文
        StandardEvaluationContext evalContext = new StandardEvaluationContext();
        
        // 设置变量
        for (Map.Entry<String, Object> entry : context.entrySet()) {
            evalContext.setVariable(entry.getKey(), entry.getValue());
        }
        
        // 禁止危险操作
        evalContext.setTypeLocator(typeName -> {
            throw new EvaluationException("类型访问被禁止");
        });
        
        // 解析并执行表达式
        Expression expression = parser.parseExpression(condition);
        return expression.getValue(evalContext, Boolean.class);
    } catch (Exception e) {
        log.error("条件表达式执行失败: {}", condition, e);
        return false;
    }
}
```

```java
// 删除Service层的历史记录代码
```
