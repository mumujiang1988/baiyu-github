# 字典配置快速修改指南

**目标**: 修改 JSON 配置以使用新的复用接口  
**预计时间**: 1 小时  
**涉及文件**: `business.config.template.json`

---

## 🎯 修改步骤

### Step 1: 打开配置文件

**文件路径**: 
```
baiyu-web/src/views/erp/pageTemplate/configs/business.config.template.json
```

---

### Step 2: 找到 dictionaryConfig 部分

定位到第 905-998 行左右的 `dictionaryConfig` 配置

---

### Step 3: 逐个修改字典配置

#### 模板格式

**新格式**（复用方案）:
```json
{
  "字典名称": {
    "type": "dynamic",
    "tableName": "表名",
    "queryConfig": {
      "conditions": [...],
      "orderBy": [...]
    },
    "fieldMapping": {
      "labelField": "显示字段",
      "valueField": "值字段"
    },
    "config": {
      "api": "/erp/engine/dictionary/字典名称/data?moduleCode={moduleCode}"
    }
  }
}
```

**旧格式**（需要替换）:
```json
{
  "字典名称": {
    "type": "dynamic",
    "config": {
      "api": "/erp/engine/dictionary/字典名称?moduleCode={moduleCode}",
      "labelField": "...",
      "valueField": "..."
    }
  }
}
```

## 📝 具体配置

### 1. 销售员字典 (salespersons)

**修改为**:
```json
{
  "salespersons": {
    "type": "dynamic",
    "tableName": "sys_user",
    "queryConfig": {
      "conditions": [
        {
          "field": "deleted",
          "operator": "isNull"
        }
      ],
      "orderBy": [
        {
          "field": "nick_name",
          "direction": "ASC"
        }
      ]
    },
    "fieldMapping": {
      "labelField": "nick_name",
      "valueField": "user_id"
    },
    "config": {
      "api": "/erp/engine/dictionary/salespersons/data?moduleCode={moduleCode}",
      "labelField": "nickName",
      "valueField": "fseller",
      "ttl": 600000
    }
  }
}
```

**说明**:
-  表名：`sys_user`
-  条件：`deleted IS NULL`
-  排序：按昵称升序
-  字段映射：`nick_name` → label, `user_id` → value

## 🐛 常见问题

### 问题 1: 返回空数组

**原因**: 表名或字段名错误

**解决**: 
1. 检查 `tableName` 是否正确
2. 检查字段名是否在数据库中存在
3. 查看后端日志

---

### 问题 2: 字段映射失败

**原因**: `labelField` 或 `valueField` 配置错误

**解决**:
1. 检查字段名是否与数据库一致
2. 检查大小写是否匹配

---

### 问题 3: 权限错误

**原因**: moduleCode 参数未传递或权限不足

**解决**:
1. 确保 URL 包含 `?moduleCode=xxx`
2. 检查用户权限配置
