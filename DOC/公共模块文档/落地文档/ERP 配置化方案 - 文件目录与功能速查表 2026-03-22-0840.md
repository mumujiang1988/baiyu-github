## 💾 数据库表结构速查

### **sys_menu (菜单表)**
```sql
-- 存储所有菜单和权限
SELECT * FROM sys_menu WHERE perms LIKE '%saleOrder%';
```

### **erp_page_config (页面配置表)**
```sql
-- 存储页面 JSON 配置
SELECT config_json FROM erp_page_config WHERE module_code = 'saleOrder';
```

### **erp_approval_flow (审批流程表)**
```sql
-- 存储审批流程配置
SELECT workflow_config FROM erp_approval_flow WHERE module_code = 'saleOrder';
```

### **erp_push_relation (下推关系表)**
```sql
-- 存储下推映射配置
SELECT mapping_config FROM erp_push_relation 
WHERE source_module = 'saleOrder' AND target_module = 'deliveryOrder';
```
