- 文档中多次提到 `erp-configurable.js` 文件
- 快速上手指南要求导入该文件:
  ```javascript
  // src/router/index.js
  import erpConfigurableRoutes from './erp-configurable'
  const routes = [...routes, ...erpConfigurableRoutes]
  ```
- 认为实际文件不存在

**实际情况**:
-  **RuoYi 框架使用数据库动态路由机制**
