## 1. page.json - 页面基础配置

### 1.1 配置内容

```json
{
  "layout": "standard",
  "pageId": "saleorder",
  "pageName": "销售订单管理",
  "apiPrefix": "/erp/engine",
  "tableName": "t_sale_order",
  "permission": "k3:saleorder:query"
}
```

### 1.2 字段说明

| 字段 | 类型 | 必填 | 说明 | 示例值 |
|------|------|------|------|--------|
| `layout` | String | 是 | 页面布局类型 | `standard`（标准布局） |
| `pageId` | String | 是 | 页面唯一标识（模块编码） | `saleorder` |
| `pageName` | String | 是 | 页面显示名称 | `销售订单管理` |
| `apiPrefix` | String | 是 | API 前缀路径 | `/erp/engine` |
| `tableName` | String | 是 | 主表表名 | `t_sale_order` |
| `permission` | String | 是 | 权限标识 | `k3:saleorder:query` |

### 1.3 后端对应接口

- **通用查询接口**: `POST {apiPrefix}/query/execute`
- **字典数据接口**: `GET {apiPrefix}/dictionary/{dictKey}/data?moduleCode={moduleCode}`
- **详情配置接口**: `GET {apiPrefix}/config/detail/{moduleCode}`

## 4. dict.json - 字典配置

### 4.1 字典类型说明

系统支持两种字典类型：

1. **dynamic**: 动态字典（从数据库表查询）
2. **static**: 静态字典（固定配置数据）
3. **remote**: 远程搜索字典（带搜索功能）

### 4.2 动态字典配置示例

```json
{
  "dictKey": "customers",
  "dictType": "dynamic",
  "table": "bd_customer",
  "conditions": [
    {
      "field": "deleted",
      "operator": "isNull"
    }
  ],
  "orderBy": [
    {
      "field": "fname",
      "direction": "ASC"
    }
  ],
  "fieldMapping": {
    "valueField": "fnumber",
    "labelField": "fname"
  },
  "config": {
    "api": "/erp/engine/dictionary/customers/data?moduleCode={moduleCode}",
    "labelField": "fname",
    "valueField": "fnumber",
    "ttl": 300000
  },
  "cacheable": true,
  "cacheTTL": 300000
}
```

#### 4.2.1 字段映射说明

| 配置项 | 说明 | 示例值 |
|--------|------|--------|
| `dictKey` | 字典唯一标识 | `customers` |
| `dictType` | 字典类型 | `dynamic` |
| `table` | 数据源表 | `bd_customer` |
| `conditions` | 查询条件 | `deleted IS NULL` |
| `orderBy` | 排序规则 | `fname ASC` |
| `fieldMapping.valueField` | 值字段 | `fnumber` |
| `fieldMapping.labelField` | 标签字段 | `fname` |
| `cacheable` | 是否启用缓存 | `true` |
| `cacheTTL` | 缓存时长 (毫秒) | `300000`（5 分钟） |

### 4.3 静态字典配置示例

```json
{
  "dictKey": "orderStatus",
  "dictType": "static",
  "data": [
    { "label": "未关闭", "value": "A", "type": "success" },
    { "label": "已关闭", "value": "B", "type": "info" },
    { "label": "业务终止", "value": "C", "type": "danger" }
  ]
}
```

| 属性 | 说明 | 类型 | 可选值 |
|------|------|------|--------|
| `label` | 显示标签 | String | - |
| `value` | 实际值 | String/Number | - |
| `type` | 标签类型 | String | `success`, `info`, `warning`, `danger`, `primary` |

### 4.4 远程搜索字典配置

```json
{
  "dictKey": "nation",
  "dictType": "remote",
  "config": {
    "searchApi": "/erp/engine/dictionary/search/nation?keyword={keyword}&moduleCode={moduleCode}",
    "minKeywordLength": 1,
    "debounce": 300
  }
}
```

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `searchApi` | 搜索接口地址 | - |
| `minKeywordLength` | 最小触发关键词长度 | `1` |
| `debounce` | 防抖时间 (毫秒) | `300` |

### 4.5 字典清单汇总

| dictKey | 字典类型 | 数据源 | 用途 |
|---------|----------|--------|------|
| `salespersons` | dynamic | sys_user | 销售员选择 |
| `currency` | dynamic | bymaterial_dictionary | 币种选择 |
| `paymentTerms` | dynamic | bymaterial_dictionary | 收款条件 |
| `nation` | remote | 远程搜索 | 国家选择 |
| `tradeType` | dynamic | bymaterial_dictionary | 贸易方式 |
| `customers` | dynamic | bd_customer | 客户选择 |
| `materials` | dynamic | by_material | 物料选择 |
| `productCategory` | dynamic | bymaterial_dictionary | 产品类别 |
| `orderStatus` | static | 固定配置 | 订单状态 |
| `documentStatus` | static | 固定配置 | 单据状态 |

### 4.6 全局缓存设置

```json
{
  "globalCacheSettings": {
    "enabled": true,
    "defaultTTL": 300000
  }
}
```
