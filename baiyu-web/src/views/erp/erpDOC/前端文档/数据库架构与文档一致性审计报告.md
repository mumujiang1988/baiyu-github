## 二、详细对比分析


| searchConfig | ✅ | ❌ | 差异 | SQL拆分为search_config,文档合并到search |
| actionConfig | ✅ | ❌ | 差异 | SQL拆分为action_config,文档合并到actions |
| dictConfig | ✅ | ✅ | 一致 | 字典配置 |
| businessConfig | ✅ | ✅ | 一致 | 业务配置 |
| detailConfig | ✅ | ✅ | 一致 | 详情配置 |

## 三、不一致问题汇总

### 3.1 严重问题(必须修复)

#### 问题1: 关联字段命名不一致
- **位置:** 明细表查询配置
- **SQL:** `field: "fbillno"`
- **文档:** `field: "order_id"`
- **影响:** 查询失败
- **修复建议:** 统一使用`order_id`作为关联字段

#### 问题2: API配置缺失
- **位置:** 数据库表结构
- **SQL:** 缺少`api_config`字段
- **文档:** 包含完整的`apiConfig`
- **影响:** API方法无法配置
- **修复建议:** 添加`api_config`字段到`erp_page_config`表

#### 问题3: 模板变量不一致
- **位置:** 查询条件配置
- **SQL:** `${FBillNo}` (大写)
- **文档:** `${billNo}` (小写驼峰)
- **影响:** 变量替换失败
- **修复建议:** 统一使用小写驼峰`${billNo}`

### 3.2 中等问题(建议修复)

#### 问题4: 客户字典命名不一致
- **位置:** 字典配置
- **SQL:** `customers` (复数)
- **文档:** `customer` (单数)
- **影响:** 字典加载失败
- **修复建议:** 统一使用`customer`

#### 问题5: 客户ID字段命名不一致
- **位置:** 表单字段配置
- **SQL:** `FCustId`
- **文档:** `FCustomerId`
- **影响:** 字段映射错误
- **修复建议:** 统一使用`FCustomerId`

#### 问题6: 金额字段命名不一致
- **位置:** 表格列配置
- **SQL:** `FBillAmount`
- **文档:** `FAmount`
- **影响:** 数据显示错误
- **修复建议:** 统一使用`FBillAmount`

### 3.3 轻微问题(可选修复)

#### 问题7: 配置字段拆分方式不同
- **位置:** 整体配置结构
- **SQL:** 8个独立字段
- **文档:** 合并为单个对象
- **影响:** 无影响,前端会自动合并
- **修复建议:** 保持现状,文档说明拆分方式

#### 问题8: 操作按钮配置不完整
- **位置:** 文档配置
- **SQL:** 包含8个操作
- **文档:** 只包含4个操作
- **影响:** 文档不完整
- **修复建议:** 补充文档中的操作配置
