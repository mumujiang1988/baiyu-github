## 一、DictionaryBuilder.js 审计

### 1.1 修复内容

**问题**: get方法重复定义导致无限递归,栈溢出崩溃

**修复方案**: 将第二个get方法重命名为getData

### 1.2 代码审计

#### 修复前代码 (问题代码)

```javascript
// 第224行: 第一个get方法
get(name) {
  const builder = this.registry.get(name)
  if (!builder) {
    console.warn(` 字典未注册：${name}`)
  }
  return builder
}

// 第266行: 第二个get方法(重复定义)
get(name) { // ❌❌ 覆盖了第一个方法
  const builder = this.get(name) // ❌❌❌ 递归调用自己,导致栈溢出
  if (!builder) {
    return []
  }
  return builder.get()
}
```

#### 修复后代码 (正确代码)

```javascript
// 第224行: 第一个get方法(保持不变)
get(name) {
  const builder = this.registry.get(name)
  if (!builder) {
    console.warn(` 字典未注册：${name}`)
  }
  return builder
}

// 第266行: 重命名为getData
getData(name) { //  重命名,不再覆盖
  const builder = this.get(name) //  调用第一个get方法
  if (!builder) {
    return []
  }
  return builder.get()
}
```

### 1.3 审计结论

| 审计项 | 结果 | 说明 |
|--------|------|------|
| 方法命名 |  正确 | getData不再覆盖get方法 |
| 递归调用 |  已消除 | getData调用get,不再递归 |
| 功能完整性 |  正确 | 两个方法功能独立,互不干扰 |
| 向后兼容 |  需注意 | 调用方需修改为getData |

**审计结果**:  **通过**

**注意事项**: 
- 所有调用`engine.get(name)`获取数据的地方需改为`engine.getData(name)`
- 建议全局搜索并替换调用方

## 三、ComputedFieldEngine.java 审计

### 3.1 修复内容

**问题**: 表达式计算器运算符优先级处理错误,先处理加减法,再处理乘除法

**修复方案**: 重写表达式计算器,正确处理运算符优先级(括号 > 乘除 > 加减)

### 3.2 代码审计

**问题分析**:
- 先处理加减法,再处理乘除法,导致 `2 + 3 * 4 = 20` (错误)
- 正确结果应该是 `2 + 3 * 4 = 14`

#### 修复后代码 (正确逻辑)

```java
// 正确的优先级处理顺序
// 1. 先处理括号(最高优先级)  正确
int parenStart = expression.lastIndexOf('(');
if (parenStart >= 0) {
    int parenEnd = expression.indexOf(')', parenStart);
    if (parenEnd < 0) {
        throw new IllegalArgumentException("括号不匹配");
    }
    
    String before = expression.substring(0, parenStart);
    String inside = expression.substring(parenStart + 1, parenEnd);
    String after = expression.substring(parenEnd + 1);
    
    BigDecimal insideValue = calculateSimpleExpression(inside);
    return calculateSimpleExpression(before + insideValue + after);
}

// 2. 再处理加减法(最低优先级)  正确
int plusIndex = findOperatorIndex(expression, '+');
if (plusIndex > 0) {
    BigDecimal left = calculateSimpleExpression(expression.substring(0, plusIndex));
    BigDecimal right = calculateSimpleExpression(expression.substring(plusIndex + 1));
    return left.add(right);
}

// 3. 最后处理乘除法(高优先级)  正确
int multiplyIndex = expression.indexOf('*');
int divideIndex = expression.indexOf('/');

// 优先处理先出现的运算符(从左往右)
if (multiplyIndex > 0 && (divideIndex < 0 || multiplyIndex < divideIndex)) {
    BigDecimal left = calculateSimpleExpression(expression.substring(0, multiplyIndex));
    BigDecimal right = calculateSimpleExpression(expression.substring(multiplyIndex + 1));
    return left.multiply(right);
}
```

        if (nextMinus > 0) {

        }
