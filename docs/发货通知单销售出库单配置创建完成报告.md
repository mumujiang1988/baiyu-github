## 🔧 关键配置说明

### 1. relationConfig 配置规范
```json
// ✅ 正确（无 enabled 字段）
"relationConfig": {
  "masterTable": "xxx",
  "masterField": "xxx",
  "detailTable": "xxx",
  "detailField": "xxx",
  "operator": "eq"
}

// ❌ 错误（不要添加 enabled 字段）
"relationConfig": {
  "enabled": true,  // ← 不要这个！
  ...
}
```

### 2. queryConfig 配置规范
```json
// ✅ 必须保留 enabled 字段
"queryConfig": {
  "enabled": true,  // ← 必须保留
  "defaultConditions": [...],
  "defaultOrderBy": [...]
}
```

### 3. 字段命名规范
- **发货通知单**: PascalCase (如 `FBillNo`, `FCustomerID`)
- **销售出库单**: snake_case (如 `f_bill_no`, `f_customer_id`)
- **配置中的字段名必须与数据库表结构完全一致**
