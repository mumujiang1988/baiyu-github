### 阶段 2: erp-self-learning 增强

**核心方法论**: PDCA 持续改进循环

```
Plan → Do → Check → Act 闭环
任务执行 → 自动检查 → 经验总结 → 更新记忆/文档 → 验证效果 → 下次迭代
```

**5 个标准化阶段**:

| 阶段 | 工具/方法 | 产出 |
|------|----------|------|
| **Check** | 3 维度检查清单 | 问题识别 |
| **Act** | 5W1H+5Why+STAR | 经验沉淀 |
| **Update** | 优先级矩阵 + 标准格式 | 知识更新 |
| **Verify** | 3 种验证方法 | 效果验证 |
| **Review** | 周回顾 + 月回顾 | 持续改进 |

**新增技能创造与优化能力**:

| 能力 | 触发条件 | 流程 |
|------|----------|------|
| **技能创造** | 重复性任务≥3 次/周 | 模式识别→原型设计→测试用例→文档化 |
| **技能优化** | 满意度<4 分或成功率<85% | 描述优化/流程优化/内容增强/架构优化 |

**关键指标**:
- 📈 文件行数：491 → 819 (+67%)
- 🎯 新增能力：技能创造 + 技能优化
- ⭐ 方法论深度：从⭐⭐到⭐⭐⭐⭐⭐

3. **三类经验分类体系**
   - skill-creator: 单一经验库
   - MM: common_pitfalls / learned_skill / expert_experience

4. **PDCA 完整闭环**
   - skill-creator: 简单的迭代循环
   - MM: 标准化的 5 阶段流程 + 定期回顾

5. **技能生命周期管理**
   - 初创期 → 成长期 → 成熟期 → 衰退期
