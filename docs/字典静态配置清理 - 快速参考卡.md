## ✅ **正确的使用方式**

### **1. JSON 配置（仅支持 dynamic/remote）**
```json
{
  "dictionaries": {
    "currency": {
      "type": "dynamic",
      "config": {
        "api": "/erp/engine/dict/union/currency"
      }
    },
    "nation": {
      "type": "remote",
      "config": {
        "searchApi": "/erp/engine/country/search?keyword={keyword}&limit=20"
      }
    }
  }
}
```

### **2. 前端调用**
```javascript
// 自动加载（推荐）
onMounted(async () => {
  await preloadDictionaries()
})

// 手动获取
const currency = dictionaryManager.getDict('currency')
const options = getDictOptions('order_status')
```

### **3. 后端接口**
```bash
GET /erp/engine/dict/all              # 获取所有字典
GET /erp/engine/dict/union/{type}     # 获取单个字典
GET /erp/engine/country/search        # 搜索国家
```

## ❌ **错误的配置（会报错）**


## ⚡ **快速排查**

### **问题：页面提示 "字典未注册"**
**解决**: 将 `static` 类型改为 `dynamic` 或 `remote`

### **问题：字典数据为空**
**解决**: 
1. 检查后端数据库是否有数据
2. 执行 `window.checkDictManager()` 查看状态
3. 执行 `await dictionaryManager.loadAll(true)` 重新加载

### **问题：自定义字典如何添加？**
**解决**: 直接添加到 `sys_dict_data` 或 `bymaterial_dictionary` 表
