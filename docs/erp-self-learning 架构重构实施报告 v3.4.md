## 二、优化方案

### 设计原则

**三层架构**:
```
第一层：SKILL.md (主技能) - 快速入口，<300 行
第二层：core/SKILL.md (核心技能) - 详细指南，<600 行
第三层：references/*.md (参考资料) - 按需加载，无限制
```

**优势**:
- 🎯 **渐进式披露**: 从概要到详细，符合认知规律
- 🔍 **快速定位**: 每层职责清晰，查找效率高
- 🛠️ **易于维护**: 模块化结构，修改影响小
- 👶 **新手友好**: 快速入门 + 详细指南 + 深度参考
