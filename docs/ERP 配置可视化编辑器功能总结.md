# ERP 配置系统可视化编辑器功能总结

## 📊 功能概述

为 ERP 低代码配置系统实现了完整的可视化编辑能力，用户无需编写 JSON 代码，通过直观的 UI 界面即可完成复杂配置。

---

## ✨ 核心功能

### 1. 表单字段可视化编辑器 (custom-form-fields-editor)

**位置**: 表单配置 → 表单字段 分组

**功能特性**:
- ✅ 添加字段：点击按钮即可添加新字段
- ✅ 编辑字段：内联编辑字段标签、字段名、组件类型、是否必填
- ✅ 拖拽排序：支持拖动行调整字段顺序
- ✅ 删除字段：确认后删除，防止误操作
- ✅ 实时同步：修改立即生效并更新配置

**支持的组件类型**:
- 输入框 (input)
- 数字输入 (input-number)
- 文本域 (textarea)
- 下拉选择 (select)
- 单选框 (radio)
- 复选框 (checkbox)
- 开关 (switch)
- 日期选择 (date-picker)
- 时间选择 (time-picker)

---

### 2. 表格列可视化编辑器 (custom-table-columns-editor)

**位置**: 表格配置 → 表格列 分组

**功能特性**:
- ✅ 添加列：快速添加表格列配置
- ✅ 编辑列属性：字段名、列标题、宽度、对齐方式等
- ✅ 拖拽排序：调整列的显示顺序
- ✅ 删除列：移除不需要的列
- ✅ 实时预览：查看列配置效果

**扩展功能** (下一步实现):
- 固定列设置
- 列宽拖拽调整
- 自定义格式化
- 排序功能启用

---

### 3. 查询字段可视化编辑器 (custom-search-fields-editor)

**位置**: 查询配置 → 查询字段 分组

**功能特性**:
- ✅ 添加查询字段
- ✅ 设置字段属性 (标签、字段名、组件类型)
- ✅ 调整字段顺序
- ✅ 删除字段
- ✅ 设置默认条件

**查询配置项**:
- 字段名称
- 查询组件类型
- 是否必填
- 默认值
- 字典绑定

---

### 4. 详情页签可视化编辑器 (custom-relation-tabs-editor)

**位置**: 详情配置 → 页签配置 分组

**功能特性**:
- ✅ 添加页签：创建新的详情页签
- ✅ 设置页签类型：表格型 / 表单型
- ✅ 配置数据源：表名、API 前缀
- ✅ 详细配置对话框：
  - 表格配置：编辑表格列 JSON
  - 表单配置：编辑表单字段 JSON
  - 通用配置：页签 Key、显示选项等
- ✅ 删除页签：移除整个页签及其配置

**页签类型支持**:
- **表格型 (table)**: 展示关联数据的列表
- **表单型 (form)**: 展示单条数据的详细信息

## 📋 配置流程示例

### 配置销售订单表单字段

**步骤**:

1. 打开 ERP 配置管理页面
2. 选择目标模块 (如：saleorder)
3. 点击进入"表单配置"标签页
4. 点击左侧导航"表单字段"分组
5. 在右侧看到可视化的字段列表
6. 点击"[+ 添加字段]"按钮
7. 填写字段信息:
   - 字段标签：客户名称
   - 字段名：customerName
   - 组件类型：下拉选择
   - 必填：是
8. 点击"保存"完成添加
9. 拖拽行调整字段顺序
10. 所有修改自动保存到配置

**生成的配置数据**:
```json
{
  "fields": [
    {
      "id": 1712048400000,
      "field": "customerName",
      "label": "客户名称",
      "component": "select",
      "required": true
    }
  ]
}
```

## 💡 最佳实践

### 1. 字段命名规范
- ✅ 使用小写字母和下划线
- ✅ 遵循数据库字段命名规则
- ✅ 避免使用保留字

### 2. 组件选择建议
- 短文本 → 输入框 (input)
- 长文本 → 文本域 (textarea)
- 数值 → 数字输入 (input-number)
- 有限选项 → 下拉选择 (select)
- 是/否 → 开关 (switch)
- 日期 → 日期选择 (date-picker)

### 3. 必填项设置
- 数据库 NOT NULL 字段 → 设为必填
- 业务关键字段 → 建议必填
- 可选字段 → 不勾选必填

### 4. 字段排序原则
- 关键字段靠前 (编号、名称)
- 常用字段优先
- 逻辑相关字段相邻
