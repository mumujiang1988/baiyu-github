## 🎯 最佳实践建议

### 1. 字典接口使用规范

**✅ 推荐做法**:
```javascript
// 使用 /data 接口直接从数据库查询
const response = await request({
  url: '/erp/engine/dictionary/salespersons/data',
  method: 'get',
  params: { moduleCode: 'saleorder' }
})
```

**❌ 避免做法**:
```javascript
// 不要使用 /load 或 /search 接口（已废弃）
const response = await request({
  url: '/erp/engine/dictionary/salespersons/load',
  method: 'post'
})
```

### 2. 静态字典配置

对于固定的选项（如订单状态），使用静态配置：

```json
{
  "orderStatus": {
    "type": "static",
    "data": [
      {"label": "未关闭", "value": "A"},
      {"label": "已关闭", "value": "B"},
      {"label": "业务终止", "value": "C"}
    ]
  }
}
```

### 3. 远程搜索配置

对于需要搜索的字典（如国家），使用远程配置：

```json
{
  "nation": {
    "type": "remote",
    "config": {
      "searchApi": "/erp/engine/dictionary/search/nation?keyword={keyword}",
      "minKeywordLength": 1,
      "debounce": 300
    }
  }
}
```
